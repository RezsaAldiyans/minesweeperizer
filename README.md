# Minesweeperizer

Turn a video into a minesweeper.

## Compile Instruction

```
mkdir build
cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
make
```

To run the program, run these command

```
./minesweeperizer --file ABSOLUTE_PATH_OF_VIDEO
```
